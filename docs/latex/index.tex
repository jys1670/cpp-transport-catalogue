This repository is a simple implementation of information storage, which can potentially be used in some kind of navigation process. As of now, there is only JSON interface capable of printing/visualizing stops and routes information as well as finding the fastest path between two stops.\hypertarget{index_autotoc_md1}{}\doxysubsection{Structure}\label{index_autotoc_md1}

Preparation\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/jys1670/cpp-\/transport-\/catalogue.git}
\DoxyCodeLine{cd cpp-\/transport-\/catalogue}
\DoxyCodeLine{mkdir build \&\& cd build}

\end{DoxyCode}


Building and running main executable\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/DCMAKE\_BUILD\_TYPE=Release ..}
\DoxyCodeLine{cmake -\/-\/build . -\/-\/config Release -\/-\/target main}
\DoxyCodeLine{cd bin}
\DoxyCodeLine{./main}

\end{DoxyCode}


Building and running unit tests (requires \href{https://www.boost.org/}{\texttt{ Boost}})\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/DCMAKE\_BUILD\_TYPE=Release -\/DENABLE\_TESTING=ON ..}
\DoxyCodeLine{cmake -\/-\/build . -\/-\/config Release -\/-\/target unit\_tests}
\DoxyCodeLine{cd bin}
\DoxyCodeLine{./unit\_tests}

\end{DoxyCode}
